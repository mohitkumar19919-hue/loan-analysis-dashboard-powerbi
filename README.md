# loan-analysis-dashboard-powerbi 

## Overview

This project is an interactive Loan Analysis Dashboard built in Power BI to analyze loan applications, applicant demographics, income patterns, credit history, and property area distributions. The dashboard transforms raw loan data into meaningful insights through KPIs, charts, and visual storytelling.

## Objectives

* Understand applicant demographics and income distribution.
* Compare loan amounts across different property areas.
* Provide actionable insights through interactive visualizations.

## Dashboard Features

### Key Performance Indicators (KPIs)

* Total Applications
* Total Loan Amount
* Average Applicant Income

### Visualizations

* Gender Distribution Analysis
* Marital Status Distribution
* Education-wise Applicant Analysis 
* Total Loan Amount by Property Area

### Interactive Filters

* Property Area
* Gender

## Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Power Query
* Data Modeling
* Data Visualization

## Key Insights

* Urban areas account for the highest share of loan applications.
* Graduate applicants contribute significantly to total applications.
* Higher-income applicants generally request larger loan amounts.
* Male applicants constitute a larger proportion of the applicant base.
